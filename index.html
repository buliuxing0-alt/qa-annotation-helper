📝 标注出题修复器 v6.1

粘贴表格数据 → 智能处理空列 → 格式化输出 → 复制发给 AI

💡 使用方法：
1. 在表格里选中所有行 → 复制 (Ctrl+C)
2. 粘贴到下方文本框 (Ctrl+V)
3. 点「格式化」→ 自动处理空列
4. 复制结果 → 发给 AI 分析

📌 各类型出题核心结构

A 类：声音提到的X + 画面显示的Y → 关联判断
B 类：当前状态 vs 早前状态 → 对比（跨度≥5分钟）
D 类（50%核心）：画面事物 + 历史回顾 + 外部搜索 → 多层叠加
E 类：事件触发 → 直接复述（10分钟内）

❌ 常见错误

  • A类：纯看画面或纯听声音就能答
  • B类：回顾时间跨度 < 5分钟
  • D类：信息在视频里已讲过 / 不需要搜索
  • E类：触发事件超过10分钟
  • 所有类型：答案偏主观 / 指代不明确
