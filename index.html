A 类 - 实时多模态感知模板

A 声音+画面关联

解说/演讲者提到的内容 + 画面中出现的内容，判断是否是同一事物
Q：解说/演讲者刚提到的 [X] 和现在画面中出现的 [Y] 是同一个吗？
A：是的 / 不是，[X] 是...，[Y] 是...

A PPT内容与演讲者关联

演讲者介绍的某个人物与PPT上展示的某个概念的对应关系
Q：[演讲者] 刚介绍的这个人跟 PPT 上展示的 [大模型/产品] 是什么关系？
A：[演讲者] 是 [产品] 的 [负责人/创造者/使用者]...

B 类 - 历史回顾模板

B 比赛累计统计回顾

在比赛进行中，回顾截止当前的累计数据
Q：[当前时间点] 截止当前，场上 [得分最高/助攻最多/篮板最多] 的球员是谁？
A：[球员名]，[数据]。数据来源：[比赛统计数据]

B 发布会内容回顾

在发布会快结束时，回顾发布会开头提到的内容
Q：[当前时间点] 这场发布会 [演讲者] 提到的第一个 [友商/产品/技术] 是谁？
A：[答案]。提到时间：[早期时间点]，跨度约 [X] 分钟

B 比赛开局回顾

在比赛后期回顾比赛刚开始时发生的事
Q：[当前时间点] 这场比赛刚开始时是哪名球员 [投进第一个球/完成第一次得分]？
A：[球员名]。发生时间：[早期时间点]，跨度约 [X] 分钟

D 类 - 工具调用模板（核心 50%）

D 品牌/公司外部信息

画面中出现品牌logo → 问公司的外部信息（创立时间、创始人、母公司等）
Q：帮我查下目前画面中出现的这个品牌/公司，它的 [创立年份/创始人/母公司/上市时间] 是什么？
A：[答案]。来源：[维基百科/官网链接]

D 产品竞品对比

画面中出现某产品 → 问与竞品的对比
Q：帮我查下眼前这款产品和 [竞品名] 在 [某配置/某功能] 上有什么区别？
A：[对比结果]。来源：[评测网站/官网]

D 人物背景查询

画面中出现某人物 → 问其外部背景信息
Q：帮我查下目前画面中出现的这个人，他 [退役后在哪一年成为主教练/获得过哪些奖项/生日是哪天]？
A：[答案]。来源：[维基百科]

D 股价/市值查询

画面中出现某公司 → 问其股价或市值信息
Q：帮我查下现在屏幕展示的这个赞助商，在 [某年某月某周] 股价的涨幅/跌幅/收盘价是多少？
A：[答案]。来源：[财经网站]

D 景点/地点信息查询

画面中出现某景点/建筑 → 问其外部信息
Q：帮我查下刚才路过的那个景点旁边那家咖啡店的开创时间和创始人是谁？
A：[答案]。来源：[地图/点评网站]

E 类 - 主动交互模板

E 比赛事件触发

特定比赛事件发生时触发提醒
Q：一旦 [球员名] 投进三分了立刻提醒我 [球员名] 投进三分了
A：[球员名] 投进三分了

E 发布会/节目环节触发

发布会或节目的特定环节开始时触发提醒
Q：下半场/下一个环节开场的时候立刻告诉我 "下半场开始了"/"下一个环节开始了"
A：下半场开始了 / 下一个环节开始了

E 比分/数据触发

比分达到某个数值时触发提醒
Q：一旦比分达到 [XX:XX] 立刻提醒我比分达到 [XX:XX] 了
A：比分达到 [XX:XX] 了

⚠️ 常见 Badcase 速查

  • 坏 答案没有标准 → 模型给出不同回答无法判对错
  • 坏 A 类只靠画面或只靠声音 → 必须结合两种模态
  • 坏 B 类回顾时间 < 5分钟 → 跨度越长越好
  • 坏 D 类信息在视频里讲过 → 必须是视频外的冷门信息
  • 坏 E 类触发事件超过10分钟 → 必须在提问后10分钟内
  • 坏 指代不明确（"这个人"）→ 要写"画面当前这个人"
  • 好 D 类在 A/B 类基础上叠加工具调用 → 指物问外
  • 好 多轮QA之间有关联性 → 形成对话连贯性
